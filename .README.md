# Cultiv_8 — Crop Intelligence AI

Cultiv_8 is an AI-powered system for plant disease detection and pest analysis using deep learning.

## Tech Stack
Python 3.11
TensorFlow / Keras
OpenCV
FastAPI
Streamlit

## Project Structure

training -> model training code
inference -> prediction scripts
analytics -> Power BI analytics
dashboard -> web dashboard
models -> trained models (not pushed)
datasets -> datasets (not pushed)
